# Logs_Api
An Api build with flask which processing logs and provide the ability to scroll over logs by the timestamp they were accepted.

## Dependencies
- **python version**: 3.8
- **installations**:
  - flask
  - flasgger 
 
## Api Endpoints:
- **specification:** http://127.0.0.1:5000/apidocs
- http://127.0.0.1:5000/show-logs/
